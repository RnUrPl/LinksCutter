📎 LinksCutter

LinksCutter is a full-stack URL shortener application that allows users to create short links, manage them, and track redirections.

It is built with a modern Node.js + Express backend and a React frontend, using MongoDB as the database.

🧠 Features
🔗 Create shortened URLs
🔄 Redirect from short links to original URLs
👤 User system (if implemented)
📊 Link tracking / analytics (click count)
🍪 Cookie-based authentication
🌐 REST API architecture
⚡ MongoDB database integration
🚀 Deployed on Render

Backend

Express.js
